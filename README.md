# Backend Foundations

> Criação de bancos de dados funcionais usando Python e SQLite3  
> Creation of functional databases using Python and SQLite3

## 📌 Sobre o projeto

Este repositório contém exemplos e exercícios práticos para **criar, manipular e consultar bancos de dados funcionais** utilizando **Python** junto com **SQLite3** — uma biblioteca leve embutida no Python.  
O foco está em **conceitos de banco de dados**, **CRUD (Create, Read, Update, Delete)** e boas práticas no uso de SQLite para projetos backend.

## 🎯 Objetivos

- aprender a estruturar bancos de dados relacionais simples;
- usar Python para interagir com SQLite3;
- implementar operações básicas e consultas;
- demonstrar exemplos didáticos e funcionais.
